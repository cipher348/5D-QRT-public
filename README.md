## üöÄ Getting Started

### Requirements

- Python ‚â• 3.8
- Core scientific stack: `numpy`, `matplotlib`, `scipy`, `sympy`
- For some visualizations: `seaborn`

Install with:

```bash
pip install numpy matplotlib scipy sympy seaborn
```

### Running a Script

Pick any script (e.g. `lattice_wavefront_propagation_simulation.py`) and run:

```bash
python lattice_wavefront_propagation_simulation.py
```

Many scripts will plot results and print key numerical findings to the console.
